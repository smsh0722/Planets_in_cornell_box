# Planets_in_cornell_box
## features
- Cornell Box 환경 구성(벽, 광원 등)
- 구 기반 행성 모델링 및 텍스처 매핑
- Elastic Collision 기반 물리 모델 구현
- 점 광원으로부터 Hard Shadow 구현
